## 🛠️ 开发环境

### 快速开始

```bash
# 克隆项目
git clone https://github.com/WalleyZhang/three_ecs.git
cd three_ecs

# 安装依赖
npm install

# 启动示例
npm run example
```

### 开发工具

- **TypeScript** - 类型安全的 JavaScript
- **Three.js** - 3D 图形库
- **Jest** - 测试框架
- **Vite** - 构建工具
